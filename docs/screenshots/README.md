# Screenshots del Dashboard IoT-IDS

Esta carpeta contiene las capturas de pantalla del sistema para documentación.

## Lista de Screenshots Requeridos

1. **01_pagina_principal.png** - Página principal con sidebar y arquitectura
2. **02_comparacion_modelos.png** - Interfaz de comparación de modelos
3. **03_comparacion_resultados.png** - Tabla con resultados de comparación
4. **04_tiempo_real.png** - Configuración de simulación en tiempo real
5. **05_tiempo_real_graficos.png** - Gráficos temporales de detecciones
6. **06_analisis_archivo.png** - Interfaz de carga de archivos CSV
7. **07_analisis_resultados.png** - Resultados del análisis batch
8. **08_metricas_overview.png** - Vista general de métricas del modelo
9. **09_metricas_confusion.png** - Matriz de confusión
10. **10_metricas_comparacion.png** - Comparación entre modelos

## Especificaciones

- **Formato**: PNG
- **Resolución**: 1920×1080 o 1280×720
- **Tamaño máximo**: 500 KB por imagen
- **Optimización**: Usar TinyPNG o similar
